# Project documentation

The original academic report can be placed in this folder using a descriptive name such as:

```text
binary-universe-report.pdf
```

Before publishing the report in a public GitHub repository, remove or redact private academic identifiers such as a student ID or enrollment number.

Recommended documentation content:

- Problem statement.
- Formal definition of Σ and Σⁿ.
- Algorithm explanation.
- Complexity analysis.
- Screenshots of execution.
- Results and conclusions.
- Known limitations.
